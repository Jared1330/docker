======= 1 ========
Що таке Docker ? 

Проблеми без докера:
- Локально чи на одному сервері працює, на іншому сервері не працює
- Використання різних версій для программ для одному сервері
- Мігрування аппликейшена на інший сервер

Workfolow Docker
DockerFile ==> Docker Build ==> Image ==> Docker run ==> Container
